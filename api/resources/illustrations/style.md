# Behavior illustration style guide

This file is the style prompt for AI-generated behavior illustrations. The
`BehaviorIllustrator` service collapses it into a single paragraph and prepends
it to every generation request, so keep it written as generation-ready prose.
Reference images dropped into `references/` are attached to every request as
canonical style samples.

Friendly flat vector illustration for a young child, in a warm and calm
children's book style. Clean simple shapes with soft rounded corners and a
uniform outline. No gradients, no textures, no 3D, no realism, no text, no
letters, no numbers. Warm palette on a plain very light background: soft warm
yellow #FFD78B as the main accent, gentle neutral tones for everything else,
one or two colors maximum besides the accent. The scene shows a small child
clearly performing the described action so a 4-6 year old instantly recognizes
it. The tone is always gentle and never scary, never angry-looking, never
shaming — even for difficult actions the child looks neutral or mildly
expressive, and the drawing stays kind. One single child character per scene
unless the action requires another person. Composition: one clear subject
centered, generous empty space around it, at most one extremely simple prop,
no background details, no visual clutter. Maximum simplicity: fewer details,
more intention.
